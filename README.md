# -OPEN-SOURCE-Snarky-Bot
Snarky Bot is an open-source project for visually programming the behavior and interactions of robotic and animatronic systems, similar to the powerful animation tools found in Bottango. While Bottango focuses on intuitive, industry-standard 3D animation tools (keyframes, timelines, etc.) for physical movements, Snarky-Bot focuses on Personality-YW
